# Housing Market Analysis using Tableau

## Project Overview

This project focuses on analyzing housing market trends using Tableau. The dataset contains detailed information about house sales, including features such as price, number of bedrooms, bathrooms, floor count, house age, and area measurements.

The objective of this project is to explore the dataset, validate its quality, and prepare it for visualization and dashboard creation in Tableau.

---

## Dataset Information

* **Dataset Name:** Transformed Housing Data
* **File Format:** CSV
* **Total Records:** 21,609
* **Total Features:** 31 columns

The dataset includes various attributes related to housing properties such as:

* Sale Price
* Number of Bedrooms
* Number of Bathrooms
* Flat Area (in Sqft)
* Lot Area (in Sqft)
* Number of Floors
* Age of House (in Years)
* Years Since Renovation
* Latitude and Longitude

---

## Data Validation Report

Before performing any visualization, the dataset was examined to ensure its quality and readiness for analysis.

### Data Quality Checks

* Verified that the dataset loads correctly in Tableau.
* Checked column headers and confirmed that all fields are properly labeled.
* Verified data types for numerical columns such as price, area, bedrooms, bathrooms, and floors.
* Checked for missing values and formatting issues.

### Missing Values

No significant missing values were detected in the dataset.

### Duplicate Records

A small number of duplicate rows (11 records) were found. Given the total dataset size, this number is negligible and does not significantly impact analysis.

## Data Preparation

The dataset was imported into Tableau and prepared for visualization.

Steps performed:

* Imported the CSV dataset into Tableau.
* Reviewed the data source tab to verify column structure.
* Confirmed correct data types for numerical and categorical fields.
* Renamed certain columns for clarity where necessary.
* Prepared the dataset for visualization and dashboard development.
